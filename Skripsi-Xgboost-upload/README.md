# Skripsi XGBoost Streamlit

Aplikasi Streamlit untuk forecasting penjualan 4 kategori produk menggunakan model XGBoost dan LightGBM.

## Jalankan Lokal

```powershell
cd "C:\Skripsi\FINAL\New"
.\myenv\Scripts\streamlit.exe run app.py
```

## Deploy Streamlit Cloud

1. Upload project ini ke GitHub.
2. Buka https://share.streamlit.io.
3. Pilih repository project.
4. Isi main file path dengan `app.py`.
5. Pilih Python `3.12`.
6. Klik `Deploy`.

Panduan detail ada di `README_DEPLOY.md`.
